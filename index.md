# 📚 ドキュメントサイトへようこそ

## はじめに

このサイトはGitHub PagesとJekyllでホストされています。

Markdownファイルを書くだけで、自動的にHTMLに変換されて公開されます。

## 機能

- 📝 **Markdownサポート** - `.md`ファイルで簡単に記述
- 🎨 **テーマ対応** - Caymanテーマを使用
- 🔄 **自動変換** - MarkdownからHTMLへ自動変換
- 🌐 **無料ホスティング** - GitHub Pagesで無料公開
