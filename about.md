# About このサイトについて

このサイトは **GitHub Pages** と **Jekyll** を使用して構築されています。

## Jekyll とは？

Jekyllは静的サイトジェネレーターで、Markdownファイルを自動的にHTMLに変換してくれます。

### 主な特徴

- 📝 Markdownで簡単に記述
- 🎨 テーマのカスタマイズが可能
- 🔄 自動ビルド＆デプロイ
- 💰 完全無料

## 使い方

### 1. Markdownファイルを作成

ファイルの先頭に以下を追加：

```markdown
---
layout: default
title: ページタイトル
---

# ここから本文
```

### 2. コミット

ファイルをリポジトリにコミットします。

### 3. 自動公開

数分待つと、自動的にHTMLに変換されて公開されます！

## Markdownの書き方

### 見出し

```markdown
# H1 見出し
## H2 見出し
### H3 見出し
```

### リスト

```markdown
- 項目1
- 項目2
  - サブ項目
```

### リンク

```markdown
[リンクテキスト](https://example.com)
```

### 画像

```markdown
![代替テキスト](画像のURL)
```

### コードブロック

````markdown
```python

def hello():
    print("Hello!")
```
````

### 強調

```markdown
**太字**
*イタリック*
~~取り消し線~~
```
